**** "Gray's Twelfth Challenge"
     DWIM : Do What I Mean
     Spec Language that:
     1) Is Easy to express designs
     2) Is Compilable
     3) Can describe all applications

     System should 'reason' about application:
     Ask about exception cases
     Ask about incomplete specification
     Not be onerous
